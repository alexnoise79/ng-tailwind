# Roadmap

This document outlines planned features and components for the ng-tailwind UI Components library.

## Form Components

- [ ] **Checkbox** - Single and multiple selection
- [ ] **Radio** - Radio button groups
- [ ] **Textarea** - Multi-line text input
- [ ] **File Upload** - File input with drag & drop
- [ ] **Range/Slider** - Numeric range input
- [ ] **Rating** - Star rating component
- [ ] **Color Picker** - Color selection input

## Data Display

- [ ] **Badge** - Status indicators and labels
- [ ] **Avatar** - User profile images with fallbacks
- [ ] **Progress Bar** - Linear progress indicators
- [ ] **Progress Circle/Spinner** - Circular progress
- [ ] **Skeleton** - Loading placeholders
- [ ] **Empty State** - Empty data states
- [ ] **Timeline** - Event timeline display
- [ ] **List** - Structured list component
- [ ] **Divider** - Visual separators

## Navigation & Layout

- [ ] **Breadcrumb** - Navigation breadcrumbs
- [ ] **Stepper/Wizard** - Multi-step forms
- [ ] **Tabs** - Tab navigation (separate from Nav)
- [ ] **Sidebar** - Collapsible sidebar navigation
- [ ] **Drawer** - Slide-out drawer component

## Feedback & Overlays

- [ ] **Popover** - Popover tooltips
- [ ] **Confirm Dialog** - Confirmation dialogs
- [ ] **Loading Overlay** - Full-screen loading states
- [ ] **Snackbar** - Alternative to toast (bottom positioning)

## Advanced Components

- [ ] **Data Grid** - Advanced table with virtualization
- [ ] **Tree View** - Hierarchical tree structure
- [ ] **Carousel/Slider** - Image/content carousel
- [ ] **Calendar** - Full calendar view (not just datepicker)
- [ ] **Chart** - Basic chart components (bar, line, pie)
- [ ] **Code Editor** - Syntax-highlighted code editor
- [ ] **Rich Text Editor** - WYSIWYG editor
- [ ] **Image Gallery** - Lightbox image gallery

## Utility Components

- [ ] **Backdrop** - Reusable backdrop component
- [ ] **Scrollbar** - Custom scrollbar styling
- [ ] **Splitter** - Resizable panel splitter
- [ ] **Resizable** - Resizable containers
- [ ] **Draggable** - Drag and drop functionality

## Theme & Styling

- [ ] **Theme Switcher** - Built-in theme toggle component
- [ ] **Dark Mode Toggle** - Dark mode switcher
- [ ] **CSS Variables Helper** - Utility for theme customization

## Accessibility Enhancements

- [ ] **Skip Link** - Skip to main content link
- [ ] **Focus Trap** - Enhanced focus management
- [ ] **Screen Reader Announcements** - ARIA live regions

## Performance

- [ ] **Virtual Scrolling** - For long lists
- [ ] **Lazy Loading** - Image/content lazy loading
- [ ] **Infinite Scroll** - Infinite scroll pagination

## Developer Experience

- [ ] **Form Builder** - Dynamic form generation
- [ ] **Component Playground** - Interactive component tester
- [ ] **Theme Generator** - Visual theme customization tool
