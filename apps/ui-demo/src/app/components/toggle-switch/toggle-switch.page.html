API Reference

Inputs

Property Type Default Description
ngModel boolean false Two-way binding for the toggle switch value. Supports Angular Forms integration.
size 'sm' | 'md' | 'lg' 'md' The size of the toggle switch.
label string - Optional label text displayed next to the toggle switch.
disabled boolean false Whether the toggle switch is disabled.

Features

  • Full Angular Forms integration (ControlValueAccessor)
  • Three size options: sm, md, lg
  • Optional label support
  • Disabled state support
  • Accessible with proper ARIA attributes
  • Keyboard navigation support
  • Two-way data binding with ngModel
