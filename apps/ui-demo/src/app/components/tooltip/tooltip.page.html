API Reference

Inputs

Property Type Default Description
ngtTooltip string '' Simple string content for the tooltip. Can be used as ngtTooltip="text".
content string | TemplateRef<HTMLElement> null Content of the tooltip. Can be a string or a TemplateRef for custom content. Takes precedence over ngtTooltip.
tooltipPosition 'top' | 'bottom' | 'left' | 'right' 'top' The position of the tooltip relative to the trigger element.
delay number 200 Legacy delay in milliseconds before showing the tooltip. Use showDelay for explicit show delay.
showDelay number 200 Delay in milliseconds before showing the tooltip after hover/focus. Takes precedence over delay.
hideDelay number 200 Delay in milliseconds before hiding the tooltip after mouseleave/blur.

Usage

The tooltip is a directive that can be applied to any DOM element. Apply it using the ngtTooltip attribute.

<!-- Simple string tooltip -->
<button ngtTooltip="Tooltip text" tooltipPosition="top">
  Hover me
</button>

<!-- With content input for TemplateRef -->
<button [content]="myTemplateRef" tooltipPosition="top">
  Hover me
</button>
