API Reference

Inputs

Property Type Default Description
model TimeInput null The selected time value. Accepts string (HH:mm or HH:mm:ss), Date object, or NgtTimeStruct.
disabled boolean false Whether the timepicker is disabled.
showSeconds boolean false Whether to show the second input field.
meridian boolean false Whether to use 12-hour format with AM/PM toggle.

Outputs

Event Type Description
timeSelect EventEmitter<NgtTimeStruct> Emitted when the time is changed (via buttons or input fields).

Interfaces

Interface Properties Description
NgtTimeStruct hour: number
minute: number
second?: number		 Structure representing a time with hour, minute, and optional second.
TimeInput string | Date | NgtTimeStruct | null | undefined Type for time input. Accepts string (HH:mm or HH:mm:ss), Date object, or NgtTimeStruct.
