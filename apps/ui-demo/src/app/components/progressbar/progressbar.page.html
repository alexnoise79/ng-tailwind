API Reference

Inputs

Property Type Default Description
value number 0 The current progress value. Must be between 0 and max.
max number 100 The maximum value for the progress bar.
type 'primary' | 'success' | 'info' | 'warning' | 'danger' | 'secondary' | 'light' | 'dark' | string undefined The color variant of the progress bar.
striped boolean false Whether to display a striped pattern on the progress bar.
animated boolean false Whether to animate the striped pattern. Requires striped to be true.
showValue boolean false Whether to display the percentage value inside the progress bar.
height string undefined (h-2) Custom height for the progress bar (e.g., 'h-4', 'h-6', 'h-8').
textType 'primary' | 'success' | 'info' | 'warning' | 'danger' | 'secondary' | 'light' | 'dark' | string undefined The color variant for the text displayed inside the progress bar (when showValue is true).
ariaLabel string 'progress bar' The ARIA label for accessibility.

Outputs

Event Type Description
- - The progressbar component does not emit any events.

Host Attributes

Attribute Value Description
role "progressbar" ARIA role for accessibility.
aria-valuemin 0 Minimum value for ARIA.
aria-valuemax maxValue() Maximum value for ARIA (from max input or default 100).
aria-valuenow currentValue() Current value for ARIA (clamped between 0 and max).
aria-label ariaLabelValue() ARIA label for accessibility (from ariaLabel input or default 'progress bar').
