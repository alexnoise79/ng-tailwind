API Reference

Inputs

Property Type Default Description
size 'sm' | 'md' | 'lg' 'md' Size of the password input component.
disabled boolean false Disables the password input component.
placeholder string '' Placeholder text displayed when input is empty.
showStrength boolean false Shows a password strength indicator with 5 levels (Very Weak, Weak, Fair, Good, Strong, Very Strong).

Outputs

Event Type Description
valueChange EventEmitter<string> Emitted when the password value changes.

ControlValueAccessor

The NgtPassword component implements ControlValueAccessor, making it compatible with Angular forms (both template-driven and reactive forms).

You can use it with ngModel, formControl, or formControlName.
