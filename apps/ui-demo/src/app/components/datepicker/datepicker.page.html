Inputs

Property Type Default Description

maxDate NgtDateStruct | null null The maximum selectable date.
startDate NgtDateStruct | null null The initial month/year to display when no date is selected.
markDisabled ((date: NgtDateStruct) => boolean) | undefined undefined Custom function to mark specific dates as disabled.

Event Type Description
