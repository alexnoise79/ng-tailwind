API Reference

Inputs

Property Type Default Description
model string | Date | NgtDateStruct | null null The selected date value. Accepts ISO string, Date object, or NgtDateStruct.
disabled boolean false Whether the datepicker is disabled.
minDate DateInput null The minimum selectable date. Accepts ISO string, Date object, or NgtDateStruct.
maxDate DateInput null The maximum selectable date. Accepts ISO string, Date object, or NgtDateStruct.
startDate DateInput null The initial month/year to display when no date is selected. Accepts ISO string, Date object, or NgtDateStruct.
markDisabled ((date: NgtDateStruct) => boolean) | undefined undefined Custom function to mark specific dates as disabled.
showTime boolean false Whether to show time selection (hour, minute, second).
format 'iso' | 'iso-local' | 'date' | 'datetime' 'iso' Output format for the date string. 'iso' (UTC), 'iso-local' (local time), 'date' (YYYY-MM-DD), 'datetime' (YYYY-MM-DD HH:mm:ss).

Outputs

Event Type Description
dateSelect EventEmitter<string> Emitted when a date is selected. Returns a formatted date string based on the format option.
navigate EventEmitter<{ current: { year: number; month: number }; prev: { year: number; month: number } }> Emitted when the month/year view is navigated.

Interfaces

Interface Properties Description
NgtDateStruct year: number
month: number
day: number
hour?: number
minute?: number
second?: number		 Structure representing a date with year, month, day, and optional time components.
DateInput string | Date | NgtDateStruct | null | undefined Type for date input. Accepts ISO string, Date object, or NgtDateStruct.
DateFormat 'iso' | 'iso-local' | 'date' | 'datetime' Type for output date format options.
