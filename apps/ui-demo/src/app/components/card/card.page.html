API Reference

Inputs

Property Type Default Description
variant 'default' | 'bordered' | 'elevated' 'default' The visual style variant of the card.

Templates

Content Projection Description
[header] Header content projected using select="[header]"
[title] Title content projected using select="[title]"
[subtitle] Subtitle content projected using select="[subtitle]"
[footer] Footer content projected using select="[footer]"
ng-content Main card content. Can contain any HTML or Angular components.
