API Reference

NgtDropdown - Inputs

Property Type Default Description
placement 'bottom-start' | 'bottom-end' | 'top-start' | 'top-end' 'bottom-start' The placement of the dropdown content relative to the trigger. Controls both vertical (top/bottom) and horizontal (start/end) positioning.
isOpen WritableSignal<boolean> signal(false) A signal that controls whether the dropdown is open or closed.

Directives

Directive Selector Description
NgtDropdownTrigger [ngt-dropdown-trigger] Applied to the element that triggers the dropdown. Clicking this element will toggle the dropdown.
NgtDropdownContent [ngt-dropdown-content] Applied to the element that contains the dropdown content. This element will be positioned relative to the trigger.

Methods

Method Description
toggle() Toggles the dropdown open/closed state.
open() Opens the dropdown.
close() Closes the dropdown.
