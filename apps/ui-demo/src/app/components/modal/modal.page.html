API Reference

Inputs

Property Type Default Description
isOpen boolean | WritableSignal<boolean> false Controls whether the modal is open or closed.
title string - Optional title text displayed in the modal header.
showFooter boolean false Whether to show the footer section. When true, content with the [footer] selector will be displayed.
closeOnBackdropClick boolean true Whether clicking the backdrop closes the modal.

Outputs

Event Type Description
closed EventEmitter<void> Emitted when the modal is closed (via backdrop click, escape key, or close button).

Templates

Content Projection Description
ng-content The main modal body content. Can contain any HTML or Angular components.
[footer] Content projected into the footer section when showFooter is true.

Features

  • Focus trapping using Angular CDK
  • Keyboard navigation (Escape key to close)
  • Backdrop click to close (configurable)
  • Body scroll prevention when open
  • Smooth animations
  • ARIA attributes for accessibility
