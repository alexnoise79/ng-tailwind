API Reference

Inputs

Property Type Default Description
type 'text' | 'number' | 'email' | 'tel' 'text' Input type.
size 'sm' | 'md' | 'lg' 'md' Size of the input component.
disabled boolean false Disables the input component.
placeholder string '' Placeholder text displayed when input is empty.
showClear boolean false Shows a clear button to reset the input value.
mask string | null null Mask pattern for formatting input display. Use 9 for digits, a for letters, * for any character. Other characters are inserted as literals. Model value remains unformatted.
chip string | RegExp | null null Separator for displaying chips inside the input (string or regex pattern).
chipFormat string ',' How chip values are separated in the model value.
filter string[] | RegExp | null null Array of allowed characters or regex pattern to filter input.
mode 'decimal' | 'currency' | null null Number mode (only for type="number").
currency string 'USD' Currency code when mode="currency".
completeMethod ((query: string) => Promise<unknown[]> | unknown[]) | null null Function to call when querying for autocomplete suggestions. Can return a Promise or an array directly.
minQueryLength number 1 Minimum number of characters required before triggering autocomplete search.
delay number 300 Delay in milliseconds before triggering autocomplete search (debounce).
scrollHeight string '200px' Maximum height of the autocomplete suggestions panel.
optionLabel string | ((item: unknown) => string) | null null Property name or getter function to use as the label of an option. If not provided, the item itself is used as the label.
optionValue string | ((item: unknown) => string) | null null Property name or getter function to use as the value of an option. If not provided, the item itself is used as the value.

Outputs

Event Type Description
valueChange EventEmitter<string | number> Emitted when the input value changes.
itemSelect EventEmitter<AutoCompleteSelectEvent> Callback to invoke when a suggestion is selected. Event contains originalEvent and value.

Templates

Template Context Description
#item let-item Custom template for each suggestion item in the autocomplete panel.
#loader - Custom template for the loading state displayed while fetching suggestions.

ControlValueAccessor

The NgtInput component implements ControlValueAccessor, making it compatible with Angular forms (both template-driven and reactive forms).

You can use it with ngModel, formControl, or formControlName.
