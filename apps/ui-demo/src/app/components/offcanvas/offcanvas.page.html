API Reference

Inputs

Property Type Default Description
isOpen boolean | WritableSignal<boolean> false Controls whether the offcanvas is open or closed.
title string - Optional title text displayed in the offcanvas header.
position 'start' | 'end' | 'top' | 'bottom' 'end' The position from which the offcanvas slides in.
showHeader boolean true Whether to show the header with title and close button.
backdrop boolean true Whether to show a backdrop that dims the background.
closeOnBackdropClick boolean true Whether clicking the backdrop closes the offcanvas.

Outputs

Event Type Description
closed EventEmitter<void> Emitted when the offcanvas is closed (via backdrop click, escape key, or close button).

Templates

Content Projection Description
ng-content The main offcanvas body content. Can contain any HTML or Angular components.

Features

  • Four position options: start, end, top, bottom
  • Focus trapping using Angular CDK
  • Keyboard navigation (Escape key to close)
  • Backdrop click to close (configurable)
  • Body scroll prevention when open
  • Smooth slide animations
  • ARIA attributes for accessibility
