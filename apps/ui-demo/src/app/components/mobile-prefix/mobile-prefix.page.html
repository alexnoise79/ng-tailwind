ControlValueAccessor

The MobilePrefix component implements ControlValueAccessor, making it compatible with Angular forms (both template-driven and reactive forms).

The component accepts and returns an IMobilePrefix object containing the phone number and selected country, or a string in the format +{dialCode}{phone}.
