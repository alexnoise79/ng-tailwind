ControlValueAccessor

The MobilePrefix component implements ControlValueAccessor, making it compatible with Angular forms (both template-driven and reactive forms).

Input Format: The component accepts both formats when writing values (e.g., when patching forms):

  • String format: "+{{ '{' }}dialCode{{ '}' }}{{ '{' }}phone{{ '}' }}" (e.g., "+1234567890")
  • Object format: IMobilePrefix object with phone and country properties
  • Partial object: Can accept partial objects with only phone or only country

Output Format: The output format is controlled by the returnAsObject input:

  • When returnAsObject is false (default): Returns a string in the format "+{{ '{' }}dialCode{{ '}' }}{{ '{' }}phone{{ '}' }}"
  • When returnAsObject is true: Returns an IMobilePrefix object
