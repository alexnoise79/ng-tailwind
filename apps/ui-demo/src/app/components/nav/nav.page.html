API Reference

NgtNav - Inputs

Property Type Default Description
orientation 'horizontal' | 'vertical' 'horizontal' The orientation of the navigation items.
style 'tabs' | 'pills' | 'underline' 'tabs' The visual style of the navigation.
align 'start' | 'center' | 'end' | 'justified' 'start' The alignment of navigation items (horizontal only).
activeId string | null null The ID of the currently active navigation item.

NgtNavItem - Inputs

Property Type Default Description
label string - Required. The label text displayed in the navigation item.
disabled boolean false Whether the navigation item is disabled.
routerLink string | string[] | null null Optional Angular Router link. When provided, the item will use router navigation and sync with RouterLinkActive.

NgtNavItem - Outputs

Event Type Description
activated EventEmitter<void> Emitted when the navigation item becomes active (only when routerLink is not used).

Templates

Component Content Projection Description
NgtNavItem ng-content Custom content for the navigation item. If provided, it will be used instead of the label.
