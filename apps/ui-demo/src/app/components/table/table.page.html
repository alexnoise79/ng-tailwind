API Reference

Inputs

Property Type Default Description
value unknown[] [] An array of objects to display in the table.
columns TableColumn[] [] Array of column definitions. Each column has field, header, and optional sortable property.
size 'sm' | 'md' | 'lg' 'md' Size of the table cells.
showGridlines boolean false Whether to show borders between cells.
striped boolean false Whether to apply alternating row colors.
paginator boolean false Whether to show pagination controls.
rows number 10 Number of rows to display per page.
reorderableColumns boolean false Whether columns can be reordered by dragging.
sortMode 'single' | 'multiple' 'single' Whether to allow single or multiple column sorting.

Outputs

Event Type Description
sortChange EventEmitter<{ field: string; order: SortOrder }> Emitted when the sort order changes.
pageChange EventEmitter<{ page: number; first: number; rows: number }> Emitted when the page changes (only when paginator is enabled).
columnReorder EventEmitter<{ columns: TableColumn[]; dragIndex: number; dropIndex: number }> Emitted when columns are reordered (only when reorderableColumns is enabled).

Templates

Template Context Description
#header columns: TableColumn[] Custom template for the table header row.
#body row: unknown, rowIndex: number, columns: TableColumn[], trClasses: string Custom template for each table body row.
#footer columns: TableColumn[] Custom template for the table footer row.
#caption - Custom template for the table caption.
