API Reference

Inputs

Property Type Default Description
options Record<string, unknown>[] | unknown[] [] Array of options to display. Can be primitives, objects, or key-value pairs.
optionLabel string 'label' Property name to use as label for object options.
optionValue string 'value' Property name to use as value for object options.
optionDisabled string 'disabled' Property name to check for disabled state in object options.
optionGroup string 'group' Property name to use for grouping options.
disabled boolean false Disables the select component.
placeholder string '' Placeholder text displayed when no option is selected.
size 'sm' | 'md' | 'lg' 'md' Size of the select component.
checkmark boolean false Shows a checkmark next to the selected option in single-select mode.
showClear boolean false Shows a clear button to reset the selection.
multiselect boolean false Enables multiple selection mode.
group boolean false Enables grouping of options.
filter boolean false Enables filtering/searching of options.
filterBy string '' Property name to filter by when using object options. If not specified, filters by the label.
fluid boolean false Makes the select component take full width of its container.
invalid boolean false Marks the select as invalid (useful for form validation).
editable boolean false Makes the select editable (allows typing to filter).

Outputs

Event Type Description
selectionChange EventEmitter<unknown> Emitted when the selection changes. Emits the selected value(s).
filterChange EventEmitter<string> Emitted when the filter text changes (only when filter is enabled).

Templates

Template Context Description
#item option: SelectOption Custom template for rendering each option in the dropdown.
#selectedItem option: SelectOption Custom template for rendering the selected item in the trigger.
#group group: SelectGroup Custom template for rendering group headers (only when group is enabled).
#header - Custom template for the dropdown header.
#footer - Custom template for the dropdown footer.
#emptyMessage - Custom template for the empty state message when no options are found.
#dropdownicon - Custom template for the dropdown arrow icon.

ControlValueAccessor

The NgtSelect component implements ControlValueAccessor, making it compatible with Angular forms (both template-driven and reactive forms).

You can use it with ngModel, formControl, or formControlName.
