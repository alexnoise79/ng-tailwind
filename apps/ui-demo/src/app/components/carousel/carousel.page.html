API Reference

Inputs

Property Type Default Description
activeId string | WritableSignal<string> - The slide id that should be displayed initially. For subsequent interactions use methods select(), next(), etc. and the (slid) output.
animation boolean true A flag to enable/disable the animations. Initialized from NgtCarouselConfig service.
interval number 5000 Time in milliseconds before the next slide is shown. Initialized from NgtCarouselConfig service.
keyboard boolean true If true, allows to interact with carousel using keyboard 'arrow left' and 'arrow right'. Initialized from NgtCarouselConfig service.
pauseOnFocus boolean true If true, will pause slide switching when the focus is inside the carousel. Initialized from NgtCarouselConfig service.
pauseOnHover boolean true If true, will pause slide switching when mouse cursor hovers the slide. Initialized from NgtCarouselConfig service.
showNavigationArrows boolean true If true, 'previous' and 'next' navigation arrows will be visible on the slide. Initialized from NgtCarouselConfig service.
showNavigationIndicators boolean true If true, navigation indicators at the bottom of the slide will be visible. Initialized from NgtCarouselConfig service.
wrap boolean true If true, will 'wrap' the carousel by switching from the last slide back to the first. Initialized from NgtCarouselConfig service.

Methods

Method Signature Description
select select(slideId: string, source?: NgtSlideEventSource) Navigates to a slide with the specified identifier.
prev prev(source?: NgtSlideEventSource) Navigates to the previous slide.
next next(source?: NgtSlideEventSource) Navigates to the next slide.
pause pause() Pauses cycling through the slides.
cycle cycle() Restarts cycling through the slides from start to end.
focus focus() Set the focus on the carousel.

Outputs

Event Type Description
slid EventEmitter<{ activeId: string; source: NgtSlideEventSource }> An event emitted when the slide transition is finished.

Features

  • Keyboard navigation (Arrow Left/Right keys)
  • Auto-cycling with configurable interval
  • Pause on hover and focus
  • Navigation arrows and indicators
  • Wrap around at ends
  • Smooth animations
  • ARIA attributes for accessibility
