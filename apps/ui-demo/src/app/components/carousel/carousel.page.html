API Reference

Inputs

Property Type Default Description
activeId string - The slide id that should be displayed initially.
animation 'fade' | 'slide' - Animation type. If not present, no animation (block display).
animations boolean true Enable/disable animations.
interval number 5000 Time in milliseconds before the next slide is shown.
keyboard boolean true Enable keyboard navigation (arrow left/right).
pauseOnFocus boolean true Pause slide switching when focus is inside the carousel.
pauseOnHover boolean true Pause slide switching when mouse cursor hovers the slide.
showNavigationArrows boolean true Show previous/next navigation arrows.
showNavigationIndicators boolean true Show navigation indicators at the bottom.
wrap boolean true Wrap the carousel by switching from the last slide back to the first.

Methods

Method Description
select(slideId, source) Navigates to a slide with the specified identifier.
prev(source) Navigates to the previous slide.
next(source) Navigates to the next slide.
pause() Pauses cycling through the slides.
cycle() Restarts cycling through the slides.
focus() Set the focus on the carousel.

Outputs

Event Type Description
slid {{ '{' }} activeId: string, source: NgtSlideEventSource {{ '}' }} Emitted when the slide transition is finished.
