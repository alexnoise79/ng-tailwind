API Reference

NgtToastService - Methods

Method Parameters Returns Description
show message: NgtToastMessage string Displays a toast notification. Returns the toast ID.
remove toastId: string void Removes a specific toast by ID.
clear - void Removes all currently visible toasts.

NgtToastMessage Interface

Property Type Required Description
severity 'success' | 'info' | 'warning' | 'danger' No The severity/type of the toast. Defaults to 'info'.
summary string No The title/summary text of the toast.
detail string No The detail/description text of the toast.
text string No Simple text message (alternative to summary/detail).
sticky boolean No Whether the toast should not auto-close. Defaults to false.
delay number No Delay in milliseconds before auto-closing. Defaults to 3000. Ignored if sticky is true.
closable boolean No Whether to show a close button. Defaults to true.
icon string No Custom SVG path data for the icon.
