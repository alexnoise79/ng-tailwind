API Reference

NgtAccordion - Inputs

Property Type Default Description
multiOpen boolean | WritableSignal<boolean> false Whether multiple accordion items can be open at the same time.

NgtAccordionItem - Inputs

Property Type Default Description
itemId string auto-generated Optional. Unique identifier for the accordion item. Auto-generated if not provided.
collapsed boolean true Whether the accordion item is initially collapsed.
disabled boolean false Whether the accordion item is disabled. When disabled, the item cannot be toggled.

Directives

Directive Selector Description
NgtAccordion [ngtAccordion] Main accordion container directive. Can be used with template reference: #accordion="NgtAccordion"
NgtAccordionItem [ngtAccordionItem] Accordion item directive. Can be used with template reference: #item="NgtAccordionItem"
NgtAccordionHeader [ngtAccordionHeader] Header container directive for the accordion item.
NgtAccordionButton [ngtAccordionButton] Button directive that triggers the accordion item toggle.
NgtAccordionCollapse [ngtAccordionCollapse] Collapsible content container directive.
NgtAccordionBody [ngtAccordionBody] Body content directive with default styling.
