API Reference

Inputs

Property Type Default Description
variant 'primary' | 'outline' | 'ghost' 'primary' The visual style variant of the button.
size 'sm' | 'md' | 'lg' 'md' The size of the button.
type 'button' | 'submit' | 'reset' 'button' The HTML button type attribute.
disabled boolean false Whether the button is disabled.
loading boolean false Whether the button is in a loading state. Shows a spinner and prevents clicks.

Outputs

Event Type Description
click EventEmitter<Event> Emitted when the button is clicked. Note: clicks are prevented when the button is disabled or loading.

Templates

Content Projection Description
ng-content The button label text. Can contain any HTML or Angular components.
