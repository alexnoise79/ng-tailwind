API Reference

Inputs

Property Type Default Description
ngModel any - Two-way binding for the checkbox value. Supports Angular Forms integration via ControlValueAccessor.
label string - Optional label text displayed next to the checkbox (after by default).
labelBefore boolean false When true, displays the label before the checkbox instead of after.
size 'sm' | 'md' | 'lg' 'md' The size of the checkbox.
trueValue any true The value to use when the checkbox is checked. Allows custom values instead of boolean true.
falseValue any false The value to use when the checkbox is unchecked. Allows custom values instead of boolean false.
indeterminate boolean false When true, displays the checkbox in an indeterminate state (neither checked nor unchecked).
disabled boolean false Whether the checkbox is disabled.

Content Projection

Template Description
#trueTemplate Custom template to display when the checkbox is checked. Use ng-template #trueTemplate.
#falseTemplate Custom template to display when the checkbox is unchecked. Use ng-template #falseTemplate.

Features

  • Full Angular Forms integration (ControlValueAccessor)
  • Three size options: sm, md, lg
  • Optional label support with flexible positioning (before or after)
  • Custom true/false values support
  • Indeterminate state support
  • Disabled state support
  • Custom templates for checked/unchecked states
  • Accessible with proper ARIA attributes
  • Keyboard navigation support
  • Two-way data binding with ngModel
