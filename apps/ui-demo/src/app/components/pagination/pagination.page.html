API Reference

Inputs

Property Type Default Description
currentPage number 1 The current active page number.
totalPages number | null null The total number of pages. If null, will be calculated from totalItems and pageSize.
totalItems number | null null The total number of items. Used to calculate totalPages when totalPages is not provided.
pageSize number 10 The number of items per page. Used with totalItems to calculate totalPages.
maxVisiblePages number 5 The maximum number of visible page buttons before showing ellipsis.
showFirstLast boolean true Whether to show first and last page buttons.
showPrevNext boolean true Whether to show previous and next page buttons.
size 'sm' | 'md' | 'lg' 'md' The size of the pagination buttons.
disabled boolean false Whether the pagination is disabled.

Outputs

Event Type Description
pageChanged EventEmitter<number> Emitted when the current page changes. Emits the new page number.
